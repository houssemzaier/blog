Houssem Zaier a.k.a. HZ is an Android enthusiast and a fullstack web developer based in Paris. He has a passion for writing apps for the mobile and also for the web.

He has focused on native mobile development with Android SDK native. He worked also with cross plateform and hybride mobile framework.

When it comes to web development HZ is interrested in fullstack development: SPA (Single Page Apps) using the Angular2 as a Front-End Framework, and Java Spring Framework or AspDotNet Core as a Back-End technologie.

HZ worked also on desktop apps using the WPF Microsoft desktop Framework, so he knows how to deal with rich client apps.
